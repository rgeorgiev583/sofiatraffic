# sofiatraffic
Tools for fetching information about public transit in Sofia from the Urban Mobility Centre Web APIs.

## Installation of CLI tool

        go get github.com/rgeorgiev583/sofiatraffic/stcli

## Usage of CLI tool

        stcli -h
